# Meeting Prep Assistant

Spike 1 validates the Windows desktop foundation for the project using Tauri with a minimal React and TypeScript frontend.

## Commands

```bash
npm install
npm run tauri dev
npm run build
```

## Spike 1 Scope

This build intentionally covers only:

- Tauri application setup
- Basic desktop window
- System tray behavior
- Background hide-to-tray behavior
- Native notification test
- Notification reopen path
- Autostart feasibility

It intentionally does not include Google integration, meeting detection, AI generation, settings, or production UI polish.
